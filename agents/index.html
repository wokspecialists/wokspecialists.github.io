Chopsticks Agents

Deploy extra Agents to run simultaneous sessions across multiple voice channels in the same server.

How simultaneous sessions work

Each Agent is an independent bot user. In one server, one Agent can hold one voice connection at a time. If two channels need different audio simultaneously, deploy two Agents. More Agents equals more independent sessions.

Example

Channel A: Agent 0001 plays Track A for Group A
Channel B: Agent 0002 plays Track B for Group B

Session control options
  • Role-controlled session: one user or role manages playback and queue
  • Open queue: channel members can add tracks (with configured limits)

Invite builder

Generate a batch of OAuth invites from the Agent list. Invites are driven by data/agents.json.
