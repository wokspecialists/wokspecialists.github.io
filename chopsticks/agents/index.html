Agents

Agents are additional Chopsticks instances. They exist for one reason: concurrency.

Hardened model

  • Each Agent is an independent bot user.
  • In one server, one Agent can hold one voice connection at a time.
  • If multiple voice channels need different audio at the same time, invite multiple Agents.

Use cases

  • Two rooms, two vibes: Channel A runs Track A while Channel B runs Track B.
  • Event night: a main room plus one or more side rooms.
  • Long sessions: multiple groups cycling in/out without stepping on each other.

Invite builder

Choose how many channels you want active at once, then invite that many Agents.
