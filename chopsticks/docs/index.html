Quickstart

  1. Invite Chopsticks.
  2. If you want multiple voice channels at once, invite multiple Agents.
  3. Join a voice channel and use the voice commands.

Member rooms that spin up on demand and clean up automatically.

  • /vc rename

Two rooms at once

Invite two Agents. Each room uses its own Agent session.

Commands
  • /play
  • /queue

Agents model

  • Each Agent is an independent bot user.
  • In one server, one Agent can hold one voice connection at a time.
  • More simultaneous rooms requires more Agents invited.

Sharding (scaling note)

Sharding splits one bot into multiple gateway connections so Discord traffic and cache load are distributed. Early on you can run a single shard. As servers grow, you move to multiple shards and route events across them.
