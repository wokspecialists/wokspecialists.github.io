Chopsticks

A hosted Discord multitool. One bot for day-to-day utility, with optional subsystems that scale when your server gets busy.

Core idea

Chopsticks aims to reduce bot clutter by concentrating common workflows into one reliable tool. Some modules are live now. Others are staged and roll out as they stabilize.

Use cases

Owners

A tool that behaves predictably when many people use it at once.

Power users

Clear controls, quick commands, and fewer edge-case surprises.

Multi-room servers

When people split into rooms, voice concurrency matters. That’s where Agents help.

Agents (voice scaling)

Agents are deployable Chopsticks instances. More Agents equals more simultaneous voice sessions in the same server.

Rule
  • 1 Agent = 1 active voice connection in your server
  • 2 voice channels at once = invite 2 Agents
  • 3 voice channels at once = invite 3 Agents
