Chopsticks

A multi-agent Discord system: main bot + deployable Agents for concurrency.

What it is

Chopsticks is the orchestration layer: commands, queue rules, and coordination across multiple deployable Agents. The main bot handles control surfaces and routing. Agents do the voice work.

Core features
  • Playback and queue management
  • Controlled access patterns (DJ-controlled or open queue)
  • Predictable stop/skip behavior with grace timing
  • Multi-channel concurrency via Agents

Architecture

The main bot coordinates. Agents are separate bot users pulled from a pool. Each Agent can run one voice session per server. Multiple channels require multiple Agents.
